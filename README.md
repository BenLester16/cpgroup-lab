BenLester16-patch-1分支为本团队第二份独立作业：课题组网站

网站栏目概览
1.首页：展示课题组最新动态、公告与学术活动照片。
2.研究方向：详细介绍各研究方向的背景、研究思路与重点成果。
3.成员介绍：包含导师简介、博士后、博士与硕士生信息，可点击查看个人主页及联系方式。
4.成果展示：列出代表性论文、会议报告、专利与获奖情况，并提供下载链接。
5.学术活动：记录课题组举办的研讨会、学术沙龙、读书会等活动日志及照片。
6.生活相册：课题组记录本，提供科研生活、日常细节和运动活动等方方面面。

联系我们：包括课题组地址、联系方式、交通指引与电子邮箱，便于校内外师生与企业进行交流。
